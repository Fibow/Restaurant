# Система управления рестораном (REST API)

RESTful API, созданное на базе Spring Boot, для управления базовыми процессами ресторана: ведением списка заведений, составлением меню и обработкой клиентских заказов.

## Стек технологий
* **Язык:** Java
* **Фреймворк:** Spring Boot (Spring Web, Spring Data JPA)
* **База данных:** H2 Database (встроенная, работает в оперативной памяти)
* **Сборщик проекта:** Maven

## Основные возможности
* **Управление ресторанами:** Создание, чтение, обновление и удаление (CRUD) информации о ресторанах.
* **Управление меню:** Добавление блюд и напитков с указанием их стоимости.
* **Обработка заказов:** Создание заказов, которые могут включать сразу несколько блюд (реализована связь «Многие ко многим» в базе данных).
* **Умная бизнес-логика:** Сервер автоматически высчитывает итоговую сумму (`totalPrice`) для каждого нового заказа на основе выбранных пунктов меню.

## Как запустить проект
Использовать ссылку https://restaurant-api-bex.azurewebsites.net/ и проверять с помощью Postman.
Или
1. Склонируйте этот репозиторий на свой компьютер.
2. Откройте проект в IntelliJ IDEA (или любой другой Java IDE).
3. Запустите приложение через главный класс (Main).
4. Сервер запустится на порту `8080`, а база данных H2 инициализируется автоматически.

## Точки доступа API (Endpoints)

### 1. Рестораны
| Метод | Эндпоинт | Описание |
|--------|----------|-------------|
| GET | `/api/restaurants` | Получить список всех ресторанов |
| POST | `/api/restaurants` | Добавить новый ресторан |
| POST | `/api/restaurants/by-params` | Добавить новый ресторан с помощью params |
| PUT | `/api/restaurants/{id}` | Обновить данные ресторана по его ID |
| DELETE | `/api/restaurants/{id}` | Удалить ресторан по ID |

### 2. Меню (Блюда)
| Метод | Эндпоинт | Описание |
|--------|----------|-------------|
| GET | `/api/menu-items` | Получить список всех блюд |
| POST | `/api/menu-items` | Добавить новое блюдо в базу |

### 3. Заказы
| Метод | Эндпоинт | Описание |
|--------|----------|-------------|
| GET | `/api/orders` | Получить список всех заказов |
| POST | `/api/orders` | Создать новый заказ (сумма рассчитается автоматически) |

## Примеры JSON-запросов (для Postman)

**Добавить блюдо в меню:**
```json
POST http://localhost:8080/api/menu-items
{
  "name": "Пицца Маргарита",
  "price": 3500.0
}
```
Создать новый заказ:
```json
POST http://localhost:8080/api/orders
{
  "status": "NEW",
  "restaurantId": 1,
  "items": [
    { "id": 1 },
    { "id": 2 }
  ]
}
```
